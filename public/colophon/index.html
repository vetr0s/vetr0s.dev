The tools, themes, and influences behind this site.

Built with

  • Hugo: static site generator

Typography

  • Zenbones Brainy: a custom Iosevka build by mcchrish, “Brainy” variant inspired by JetBrains Mono and Berkeley Mono

Colors

The light and dark color schemes are based on the Modus themes by Protesilaos Stavrou:

  • modus-operandi: light
  • modus-vivendi: dark

Headings use the yellow-warmer, magenta, and cyan accents from the same palettes.

Design influence

  • gingerbill.org: the inline-list nav with | separators is borrowed from Bill’s site
